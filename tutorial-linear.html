LINEAR SEARCH

Linear Search is a searching algorithm that checks each element one by one from the beginning of the array until the target element is found.

Rules

✓ Start searching from index 0.

✓ Check elements sequentially.

✓ Do not skip any index.

✓ Stop when the target is found.

✓ You cannot directly select the target element.

Example

Find "68"
0 1 2 3 4 5 6 7
22
11
44
12
68
89
72
32
22 ❌ → 11 ❌ → 44 ❌ → 12 ❌ → 68 ✅
❌ Wrong: Clicking 68 directly.
