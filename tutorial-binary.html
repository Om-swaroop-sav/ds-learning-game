BINARY SEARCH

Binary Search works only on sorted arrays. It repeatedly finds the middle element and removes half of the search space until the target is found.

Rules

✓ Array must be sorted.

✓ Calculate Mid = (Low + High) / 2.

✓ Compare Mid with Target.

✓ Search Left or Right half.

✓ Continue until target is found.
